# SDD · Anclora Data Lab

Este repositorio adopta SDD como marco de definición y ejecución.

## Estructura

- `sdd/core/`: visión base, identidad de producto y reglas núcleo.
- `sdd/features/`: especificaciones de features.
- `public/docs/`: documentos públicos, arquitectura y roadmap.
- `.agent/`: gobernanza multiagente, reglas y skills.

## Feature activa inicial

- `datalab-foundation-v1`

## Principios

- continuidad de marca y narrativa con `Anclora Private Estates`
- capa analítica premium, no dashboard genérico
- acceso selectivo y lenguaje institucional
- producto preparado para evolución por fases
