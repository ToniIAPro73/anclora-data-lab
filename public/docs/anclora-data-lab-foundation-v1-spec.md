# Anclora Data Lab · Foundation v1

`Anclora Data Lab` nace como la capa analítica premium de `Anclora Private Estates`: una aplicación privada para señales territoriales, packs de inteligencia y activos documentales con acceso selectivo.

Esta primera versión implanta:

- identidad visual propia derivada del logo de Data Lab
- continuidad narrativa con el Área Privada de Private Estates
- login privado funcional
- workspace inicial con radar, zonas, alerts y biblioteca
- base SDD y documentación pública

La fundación v1 prepara fases posteriores con mayor densidad analítica, scoring, copiloto IA y control de acceso más granular.
