# Anclora Data Lab · Access Architecture v1

## Posicionamiento

`Anclora Data Lab` es una aplicación especializada de `Anclora Private Estates`.

## Relación con Private Estates

- `Private Estates` actúa como puerta de entrada pública y premium.
- El Área Privada anuncia Data Lab como sala de inteligencia selectiva.
- `Data Lab` materializa esa promesa mediante acceso privado y activos analíticos curados.

## Capas de acceso inicial

1. **Pública**
   - narrativa
   - promesa de producto
   - documentos fundacionales

2. **Privada**
   - señales curadas
   - radar territorial
   - packs activos
   - documentación premium

## Roles iniciales

- `datalab-admin`
- `market-analyst`
- `investment-advisory`
- `partner-intelligence`
- `investor-viewer`

## Evolución prevista

- control más fino por módulo
- separación de audiencias internas y externas
- integración futura con `Anclora Group`
